Getting Started

Bootstrap4ik is easy to integrate into your existing projects. Follow these simple steps to get started.

Installation
<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.8/dist/css/bootstrap.min.css" rel="stylesheet">
<link rel="stylesheet" href="styles.css">

Simply include the Bootstrap CSS and our custom stylesheet in your HTML file. That's it! You're ready to start building beautiful interfaces.

Components

Bootstrap4ik includes a wide range of customizable components that you can use to build your interface.

Cards

Flexible content containers with built-in hover effects and smooth animations.

Navigation

Responsive navigation bars with mobile-friendly collapsible menus.

Forms

Beautiful form controls with validation states and custom styling.

Buttons

Multiple button styles with hover effects and gradient options.

Utilities

Custom utility classes that help you build faster and more efficiently.

Color Utilities
  • --text-main - Main text color
  • --text-primary - Primary accent color
  • --bg-secondary - Secondary background
Button Utilities
  • --btn-primary - Primary button style
  • --btn-secondary - Secondary button style
